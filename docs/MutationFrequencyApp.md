# Mutation Frequency App

There are a few parts to this app:

Charts located in `src/features/charts`
- `GeneFrequencyChart`: Left chart on Genes Tab

Tables located in `src/features/GenomicTables`
- `GenesTable`
- `SomaticMutationsTable`
The main app is located in `src/features/genomic`

You can change the style by editing the appropriate component.
