Cohorts can be exported and imported using the buttons on the Cohort Manager:

![CohortExportImport](images/CohortImportAndExport.png)

Export will create a `.tsv` file containing `case_id` and `submitter_id` columns for the current cohort.
Import accepts case UUIDs or case submitter IDs entered directly or uploaded in `.txt`, `.csv`, or `.tsv` files. Exported cohort `.tsv` files can be imported directly.
